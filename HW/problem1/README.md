# Socket Homework
This homework create a server program and a cilent program to simulate communication of two computers nowadays. And they use socket to communicate.

## How To Work
1. Run local server first.  
$ python3.7 socket_server.py
2. Then run local client.  
$ python3.7 socket_client.py
3. Client inputs a math question.   
e.g. $ 3 + 7
4. Server will return an answer.
